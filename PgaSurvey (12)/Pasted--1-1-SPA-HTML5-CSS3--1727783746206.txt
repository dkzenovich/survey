Общие требования:
1.1. Разработать одностраничное веб-приложение (SPA) с использованием HTML5, CSS3 и JavaScript.
1.2. Обеспечить адаптивный дизайн для корректного отображения на десктопных и планшетных устройствах.
1.3. Использовать современные практики веб-разработки, включая семантическую разметку и доступность.
Структура интерфейса:
2.1. Верхняя панель навигации:

Логотип компании
Название системы ("Система универсального анализа")
Кнопка выхода из системы
2.2. Боковая панель:
Список предыдущих анализов
Кнопка "Новый анализ"
2.3. Основная область контента с вкладками:
Ключевые аспекты
План исследования
Собранные данные
Выводы


Функциональные требования:
3.1. Вкладка "Ключевые аспекты":

Отображение списка ключевых аспектов проблемы
Возможность добавления, редактирования и удаления аспектов

3.2. Вкладка "План исследования":

Список релевантных отделов с возможностью редактирования
Отображение вопросов для интервью по каждому отделу
Функционал редактирования и перегенерации вопросов
Отображение шагов плана исследования
Кнопки "Сохранить изменения" и "Начать исследование"

3.3. Вкладка "Собранные данные":

Таблица с данными по отделам
Фильтрация данных по отделам
Отображение статуса сбора данных
Функционал экспорта данных

3.4. Вкладка "Выводы":

Отображение основных выводов
Список рекомендаций
Визуализация приоритетности рекомендаций
Кнопки "Экспорт выводов" и "Утвердить рекомендации"


Технические требования:
4.1. HTML:

Использовать HTML5 с семантической разметкой
Обеспечить валидность HTML-кода

4.2. CSS:

Использовать CSS3 для стилизации
Применить подход Mobile First для обеспечения адаптивности
Использовать Flexbox или Grid для компоновки элементов

4.3. JavaScript:

Использовать чистый JavaScript (ES6+) без фреймворков
Реализовать модульную структуру кода
Обеспечить обработку событий для интерактивных элементов
Реализовать динамическое обновление контента без перезагрузки страницы


Интерактивные элементы:
5.1. Реализовать переключение между вкладками без перезагрузки страницы
5.2. Обеспечить функционал редактирования текстовых данных (вопросы, выводы, рекомендации)
5.3. Реализовать интерактивный график снижения продаж с возможностью наведения для получения детальной информации
5.4. Добавить всплывающие подсказки (tooltips) для пояснения функций интерфейса
